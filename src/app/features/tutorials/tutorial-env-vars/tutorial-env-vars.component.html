Ocultando Credenciais da AWS

Subir suas chaves da AWS (`Access Key` e `Secret Key`) no GitHub é uma vulnerabilidade crítica. Bots escaneiam repositórios públicos em segundos para sequestrar contas.

Sem contar que a própria AWS ao detectar sua chave em algum servidor do github, bloqueia a sua chave no mesmo instante.

Vamos aprender a esconder essas chaves usando Variáveis de Ambiente na sua IDE.

Em vez de colocar as chaves reais diretamente no arquivo, nós informamos ao Spring Boot para buscar essas informações do sistema operacional no momento da execução.

Atenção: Apague suas chaves reais do código agora mesmo!

A sintaxe ${VARIAVEL} diz ao Spring para injetar o valor automaticamente.

Se você usa o IntelliJ, as variáveis são injetadas na sua configuração de execução (Run/Debug Configurations).

No canto superior direito, ao lado do botão de "Play", clique na seta para baixo na sua aplicação e selecione Edit Configurations... Agora, quando você executar sua aplicação usando essa configuração, as chaves da AWS serão injetadas automaticamente, mantendo seu código seguro!
